# 离线开发助手

预想的功能：
1. 能够给出包名，查找并打包依赖项，提供**一揽子**打包，方便打包依赖传入离线环境使用
2. 可用根据 `verdaccio` 似有搭建的npm库，实现定期打包发布，storage，全量拆分为100以内包，增量支持按时间段或者日期增量打包
3. 一些离线和私有仓库的搭建步骤和方案总结 

初步支持的环境，`NUGET`\ `PYPI` \ `NODEJS`
